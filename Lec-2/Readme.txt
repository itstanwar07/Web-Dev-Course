Lecture 2 – Development Environment Setup

In this lecture, I set up my development environment for the course. I installed **Visual Studio Code (VS Code)** and downloaded the required extensions that will help me write, manage, and debug code more efficiently.

I also learned how to configure VS Code for web development, understand the purpose of different extensions, and organize my workspace for a better coding experience.

Although setting up the environment may seem like a small step, it is an important part of every developer's journey. A properly configured workspace makes learning and development much smoother.
