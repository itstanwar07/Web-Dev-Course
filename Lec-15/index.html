The Types of CSS

  1. Inline CSS
    2. In this the styling is done in the element tag itself
  2. Internal CSS
    3. In Internal CSS styling is done in the style tag <style> </style>;
  3. External CSS
    4. In External CSS styling is done in the different file css. Hence this is the most prefered practice, which applied in the real life projects
